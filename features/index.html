Features

What Komi Store does

A discovery layer over GitHub releases, an install layer that silently delivers updates, and an honest backend that logs nothing about you.

Get Komi Store Read the Wiki

Smart discovery

Trending, hot, and most popular GitHub projects, ranked per platform. Only repos that ship installable assets show up — zero source-only noise.

One-tap install

The right binary downloads and hands off to your OS — APK on Android, EXE/MSI on Windows, DMG/PKG on macOS, AppImage/DEB/RPM/pacman on Linux.

Silent install & auto-update

Shizuku, Sui, or Dhizuku installs without prompts on Android. Background update checks notify you; auto-update applies them while you sleep.

Signing-key verification

Auto-update is blocked the moment a new APK's signing key drifts from what the row was registered with. Prevents silent compromise on a hijacked release.

Mirror system

Multi-source race against the GitHub CDN with end-to-end SHA-256 verification against GitHub's published digests. Keeps installs working where github.com is throttled.

Library imports

Obtainium import/export in one tap. Library Imports also recognises GitHub-sourced apps already on your device — sideloaded, F-Droid, manual installs.

APK Inspect

Peek inside any release before installing — app label, signing fingerprint, version codes, min/target SDK, components, and permissions colour-coded by protection level.

Open-source backend

Apache-2.0 and self-hostable. Search queries are hashed client-side, IP addresses are not stored, and the same data flows whether you're signed in or not.

Get your app listed

No manual submission needed. Komi Store uses the public Search API — if your project meets these conditions, it appears automatically.
