Why donate

Komi Store is built and maintained by one developer. The app is free. The source code is open. The backend is open. The project pays for itself only through sponsorship.

There is no premium tier today. If we ever add one, it follows a single principle: Komi Store charges only for features that cost us money to run. Anything that runs on your device stays free.

Sponsor FAQ

Is sponsorship tax-deductible?

No — Komi Store is an individual indie project, not a registered non-profit. Sponsorship is a personal contribution to the maintainer.

Will sponsorship influence the roadmap?

Sponsors at the Supporter tier and above get early roadmap visibility and a direct channel to the maintainer. They do not get to dictate features that conflict with the project's privacy or open-source commitments. The project's commitments — no telemetry by default, no proprietary dependencies, no charging for features that run on your device — are non-negotiable.

Can my company sponsor?

Yes. Reach out to hello@github-store.org for invoicing options, custom logo placement, or partnerships beyond the standard tier ladder.

What if I can't sponsor money but want to help?

Star the repository, file issues for bugs you find, contribute translations, or submit a pull request. Code contributions are welcome under the contributor guide.
