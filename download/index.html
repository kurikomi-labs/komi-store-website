Verify your download

All Android releases are signed with this fingerprint. Auto-updates inside Komi Store reject any APK whose signing key doesn't match.

SHA-256:
B7:F2:8E:19:8E:48:C1:93:B0:38:C6:5D:92:DD:F7:BC:07:7B:0D:B5:9E:BC:9B:25:0A:6D:AC:48:C1:18:03:CA

SHA-256 digests for every desktop installer ship in the GitHub release page next to each asset. The mirror system inside Komi Store verifies them end-to-end against GitHub's published digests.
