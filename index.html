FAQ

Frequently Asked Questions

What is GitHub Store?

+

GitHub Store is a free, open-source app store for GitHub releases. It works on Android, Windows, macOS, and Linux. License: Apache-2.0.

Is GitHub Store free?

+

Yes. The app, the source code, and the backend are all open source under Apache 2.0.

How is GitHub Store different from F-Droid?

+

F-Droid only includes apps in its curated FOSS-only repository. GitHub Store discovers and installs apps directly from GitHub releases — including apps not in F-Droid.

How is GitHub Store different from Obtainium?

+

Obtainium is a power-user updater for tracked GitHub repos. GitHub Store adds a discovery layer (browse trending, popular, hot releases), an install layer (Shizuku/Dhizuku silent install), and works on Desktop too. You can import your Obtainium library with one tap.

Does GitHub Store work in China?

+

Yes. The mirror system and backend OAuth proxy work around GitHub throttling on Chinese networks.

Everything you need, nothing you don't

GitHub Store strips away the complexity of finding and installing open-source software.

Smart Discovery

Browse trending, hot releases, and most popular repos. Only apps with real installable binaries for your platform are shown — zero source-only noise.

One-Click Installs

Tap install and the correct binary downloads and hands off to your OS — APK on Android, EXE on Windows, DMG on macOS, AppImage or DEB on Linux.

Update Tracking

Remembers what you installed and highlights new releases automatically. Stay current without manually checking GitHub repos ever again.
